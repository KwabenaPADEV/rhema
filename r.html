Course Title: Statistical Analysis with R programming language for non – programmers

Course Description: R programming language is a statistical tool well-tailored for data analysis and visualization. This software has become popular among many people in academia and industry since it’s free, open source, and available on every major platform. Also, it works on common platforms such as Windows, Linux and Mac. This course provides an easy introduction to data analysis using R for those with little or no programming experience. After this course, participants will be able to use R to manipulate data and transfer their skills to other programming languages. Course aims and objectives: This intensive course aims to equip learners with in – depth understanding of how to perform basic commands in the R programming language and navigate in the RStudio interface. Participants will be provided with hands – on practical sessions on the application of R programming skills necessary for analysing data in the social sciences.

After this course, participants will be able to:

  1. Get familiar with R and R studio interactive environment
  2. Understand the syntax structure and the data types
  3. Create new variables, replace and recode values in existing variables
  4. Manipulate and wrangle data using the tidyverse package
  5. Describe data using both descriptive and inferential statistics
  6. Visualise data
  7. Analyse data

Course content:

Lesson 1
  • Introduction to R
    • Understanding the R environment, console, and script editor.
Lesson 2
  • Data exploration
    • Overview of your data structure
Lesson 3
  • Data manipulation
    • Using tidyverse package for data manipulation tasks (filtering, selecting, arranging).
Lesson 4
  • Describing your data
    • Exploring the centrality of your data and summarising the data.
Lesson 5
  • Data visualization using ggplot2
    • Introduction to the grammar of graphics and ggplot2 syntax.
Lesson 6
  • Statistical analysis
    • Introduction to statistical tests (t-tests, ANOVA) and regression analysis.

Course Prerequisites:

  • Basic computer literacy and familiarity with internet usage.
  • No prior experience with R programming language or programming, as the course covers fundamentals.
  • A genuine interest in data analysis.
