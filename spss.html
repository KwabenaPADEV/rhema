Course Title: Statistics using SPSS

Course Description: Statistical Package for Social Sciences (SPSS) is a statistical tool for quantitative analysis of complex data. This course will give you the requisite knowledge and hands – on experience using real – life examples of data provided to learn the basics. The course is designed for individuals who are new to SPSS and equip them to handle this useful analysis tool.

Course Description: This course is an intensive course that will help you to familiarize yourself with the interface and analyse your data with ease. You will be able to input data in the best format for SPSS. Also, beyond the ability to do your analysis, the course will guide you through the key principles behind the various statistical tests you are using.

After taking this course, you will be able to:

  1. Understand the SPSS environment
  2. Know how to create and import data
  3. Undertake basic data preparation and management
  4. Run descriptive statistics and regression analysis

Course content:

Lesson 1
  • Interaction with SPSS
    • Knowing the various ways of interacting with SPSS
Lesson 2
  • Data view window
Lesson 3
  • Creating data and importing data
    • Learn how to create and import data
Lesson 4
  • Running and interpreting statistical tests
    • Learn how to run descriptive, inferential and regression analysis

Course Prerequisites:

  • Basic computer literacy and familiarity with internet usage.
  • No prior experience with SPSS , as the course covers fundamentals.
  • A genuine interest in data analysis.
