Course Title: Data Analysis with STATA

Course Description: This course provides a comprehensive understanding of STATA, a statistical tool for researchers to wrangle, visualize and analyse data. The course is designed for those new to STATA and learners desiring to increase their proficiency. By the end of this course, you will be able to carry out tasks such as entering and importing data, analysing data, performing hypothesis tests and interpretating statistical results. Course aims and objectives: This interactive course is intended to build your and your understanding of the key concepts in STATA. Participants will be taken through hands – on practical projects to enable them gain fundamental knowledge in working with STATA,

completion of this course, you’ll be able to:

  1. Understand the STATA environment
  2. Import and export data from different file formats
  3. Create basic plots and work with graph editor
  4. Handle different variable types and perform statistical analysis

Course content:

Lesson 1
  • Introduction to Stata
    • Able to navigate the Stata interface
Lesson 2
  • Manipulating data
    • Create reproducible scripts in the Do-file editor
Lesson 3
  • Obtaining and managing data
    • Learn to import diverse data and merge them
Lesson 4
  • Statistical analysis
    • Learn to do descriptive and inferential analysis; statistical tests (t-tests, ANOVA) and regression analysis.

Course Prerequisites:

  • Basic computer literacy and familiarity with internet usage.
  • No prior experience with STATA , as the course covers fundamentals.
  • A genuine interest in data analysis.
